Projects

Austin Data Bass Web App

Worked with a team of 6 to develop an "IMDBlike" web app for Austin area concerts. Combined information from several different APIs and displayed it in useful and easy to use fashion. We used HTML, CSS, Bootstrap and Django templates for the front end and Django, Python and a PostgreSQL database for the backend. Created a test suite using Selenium and Python unittest . I was reponsible for leading the design of the front end. Used Git and Agile methodologies for our project managment. This was for EE461L: Software Engineering and Design Laboratory.

Pintos Operating System

Worked with a partner to add features to a rudimentary operating system of over 100,000 lines of pre-existing C code. Implemented virtual memory paging, system calls, directory support and basic file system. Worked on the project in a Linux environment and utilized Git for version control. This was for EE461S: Operating Systems.

CS50: Intro to AI with Python

Worked on 10+ assignments that implemented various artifical intelligence, machine learning and data science algorithms. Learned to use libraries like TensorFlow to create neural networks. Honed my skills with Python and practiced good code styling while focusing on runtime efficiency. This was a apart of Harvard's online edX classes and was a class I took to explore my interest in the field. Click here to see quick video demonstrations I made for the project submissions.

Book Review Web App

Created a book review web application using Flask, a PostgreSQL database, Python, HTML and CSS. The site was complete with a login system, review & rating system and an extensive database of 5,000 unique books. This was made as part of an open ended project for Harvard's CS50 Web Development class. I gained experience using APIs to pull in data and even created an API of my own to share individual book data.
